# 🚀 InternMate – Hyperlocal Internship Finder

> A modern, student-focused platform to discover and post internships based on location.

---

## 🌐 Live Demo

🔗 https://your-vercel-link.vercel.app

💻 GitHub Repository:
https://github.com/jadhav-swarali/internmate

---

## ✨ Features

* 🔍 **Search Internships** – Instantly find roles by title
* 📍 **Location Detection** – Shows opportunities near you
* ➕ **Post Internship** – Add new opportunities dynamically
* ⚡ **Real-time UI Updates** – Data updates instantly
* 🎨 **Modern UI** – Glassmorphism design with Tailwind CSS

---

## 🛠️ Tech Stack

| Technology   | Usage                |
| ------------ | -------------------- |
| React        | Frontend development |
| Tailwind CSS | UI styling           |
| Vite         | Fast build tool      |
| JSON Server  | Mock backend         |
| Vercel       | Deployment           |
